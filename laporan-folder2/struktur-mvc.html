Model berfungsi sebagai representasi data dan logika bisnis aplikasi. Setiap file merepresentasikan satu tabel di database (menggunakan Eloquent ORM Laravel), serta mengelola relasi antar tabel seperti hubungan antara User, Product, dan OrderHistory.

View bertugas menampilkan antarmuka kepada pengguna menggunakan Blade Template Engine Laravel. Folder admin/ berisi halaman khusus panel admin, folder pages/ berisi halaman yang diakses pelanggan seperti beranda, detail produk, dan checkout.
⚙️ Controller — app/Http/Controllers/
app/Http/Controllers/
├── AdminDashboardController.php
├── AdminProductController.php
├── AdminOrderController.php
├── AdminUserController.php
├── AuthController.php
├── PageController.php
├── AddressController.php
└── Controller.php
Controller bertindak sebagai jembatan antara Model dan View. Controller menerima request dari browser, memanggil Model untuk mengambil data dari database, kemudian meneruskan data tersebut ke View untuk ditampilkan. Contoh: AdminDashboardController mengambil statistik penjualan, pesanan, produk, dan pelanggan lalu mengirimnya ke halaman dashboard.
🗄️ Konfigurasi Database — config/database.php
config/database.php

'default' => 'mysql',

'mysql' => [
    'driver' => 'mysql',
    'host' => env('DB_HOST', '127.0.0.1'),
    'port' => env('DB_PORT', '3307'),
    'database' => env('DB_DATABASE', 'furninest'),
    'username' => env('DB_USERNAME', 'root'),
    'charset' => 'utf8mb4',
]
Konfigurasi Database mendefinisikan koneksi antara aplikasi Laravel dengan database MySQL. Nilai seperti host, port, nama database, dan username diambil dari file .env agar mudah disesuaikan tanpa mengubah kode. Database yang digunakan adalah furninest pada port 3307.
